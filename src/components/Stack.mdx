import Stack from "./Stack";

# Stack

The Stack component is for spacing elements out. It will evenly space all its children.

```js live
<Stack>
  <div>Orange</div>
  <div>Apple</div>
  <div>Banana</div>
</Stack>
```

## Props

```js live
<Stack as="ol" space="0.5rem">
  <li>Orange</li>
  <li>Apple</li>
  <li>Banana</li>
</Stack>
```
